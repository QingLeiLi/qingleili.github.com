Jquery笔记

应用

  1. 格式化表单数据为Json格式
/*
serializeArray()获取表单数据是数组格式
[
    {
        "name":"user",
        "value":"test"
    },
    {
        "name":"passwd",
        "value":"test"
    }
]
*/

var d = {};  //用来保存json
//获取到数组格式数据
var t = $('#form').serializeArray();
$.each(t, function() {
  d[this.name] = this.value;
});

注意

  • 表单根据name属性来提交，如果没有name属性，serializeArray（）将得到空值
