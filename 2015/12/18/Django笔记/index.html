Django笔记

Django是Python的一个web快速开发框架，也是我接触的第一个web框架，开发快，封装功能较多，安全性较好。

安装

依赖

  • Python
  • Pip

安装

  1. 直接使用pip安装

    pip install django

使用

基本命令

  • 运行网站，可指定监听的IP和端口，默认127.0.0.1:8000 

    manage.py runserver [0.0.0.0:8000]

错误汇总

  • Ajax提交设置type:POST，Django后台request.method仍为GET
    原因：urls.py设置路由时，url后面多加了一个/ 

    urlpatterns = patterns('',
    url(r'^ajax/$',ajax),
)

    解决办法：将最后一个斜杠去掉

    urlpatterns = patterns('',
    url(r'^ajax$',ajax),
)

  • 表单POST提交403错误
    原因：Django出于安全考虑，默认对POST的表单进行CSRF token校验，表单中没有token信息，就会禁止提交
    解决办法：在处理函数上添加修饰器，忽略掉该表单的CSRF验证

    from django.views.decorators.csrf import csrf_exempt

@csrf_exempt
def ajax(request):
    pass

  • 服务器返回json数据，500错误
    解决办法：序列化json，封装为HTTP response，并指定数据格式

    import json
j = json.dumps({"data":"ok"})
return HttpResponse(j, 'application/json')
