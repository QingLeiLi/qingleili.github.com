Python Xlwt笔记(读写Excel文件)

發表於 2015-12-12

Xlwt笔记

Xlwt用来对Excel文件进行简单写操作，并且可以设置单元格的样式且跨平台，在Ubuntu 12.04测试通过；但其只支持较老版本的Excel文件（97/2000/XP/2003的xls文件），不支持xlsx等office2007后加格式，如果需要，请自行百度openpyxl模块;不能读取Excel文件，如有需要，可以使用Xlrd模块，参考。

安装

依赖

  • pip

安装

pip install xlwt

其他

  • 文档

    http://xlwt.readthedocs.org/en/latest/

  • 分析excel文件的层级对象
    要读取excel的数据，就要了解excel的结构，根据excel的结构一层一层的去读取数据。

    excel有三层级对象，workbook，sheet，和cell。一个excel文件就是一个workbook，所以在最初我们必须要打开这个excel文件，也就是workbook。sheet我们都很熟悉，就是表，我们都知道一个excel文件有时候会有很多的表，所以我们必须要选择是读取哪个表的数据，最后才是cell，cell其实就是格子，excel的表格就是一个二维数组，cell就是这个表格中的最小单元，也就是我们读取数据存储的地方。

    使用

基本命令

  • 创建一个book对象

    wbk = xlwt.Workbook()

  • 创建一个工作表

    #参数cell_overwrite_ok，单元格内容是否可以被覆写
sheet1 = wbk.add_sheet('sheet 1')
sheet2 = wbk.add_sheet('sheet 2', cell_overwrite_ok=True)

  • 向单元格写入数据

    #前两个参数为单元格位置（x，y）
sheet1.write(0,1,'test text')

  • 设置单元格样式

    #获取样式对象
style = xlwt.XFStyle()
#获取字体样式  
font = xlwt.Font()
#设置字体 
font.name = 'Times New Roman'
#是否加粗  
font.bold = True
#载入设置
style.font = font
#应用
sheet1.write(0, 1, 'some bold Times text', style)

  • 保存文件（会直接覆盖同名文件）

    wbk.save('test.xls')

应用

Excel文件创建Demo

#coding=utf-8       #设置编码方式

import xlwt
import random

book = xlwt.Workbook()     #获得excel的book对象

sheet=book.add_sheet("test")#新建工作表
for row in range(0,10):
    for col in range(0,10):
        sheet.write(row,col,random.randrange(0,10))  #插入数据
book.save("test.xls")
print("Done")

错误汇总

  • Windows下没有问题，移植到Ubuntu时发生字符转码错误

    UnicodeDecodeError: ‘ascii’ codec can’t decode byte 0xe8 in position 0: ordinal not in range(128)

解决办法：在新建Excel对象时指定编码book = xlwt.Workbook(encoding = 'utf8')就可以了
