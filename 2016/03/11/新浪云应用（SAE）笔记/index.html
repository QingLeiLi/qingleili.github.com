新浪云应用（SAE）笔记

SAE为新浪的云应用服务，允许用户部署自己的应用，免费用户会有带宽和流量限制。

应用

部署Django项目

  1. 创建新应用
    需要配置二级域名和应用名，二级域名为用户访问该应用时使用的URL，如图应用使用lqlcn.applinzi.com来访问；应用名则是在应用管理中显示的应用名；环境选择Python2.7。
  2. 选择版本控制软件(Git or Svn)，得到仓库地址

  3. 在项目根目录创建应用配置文件config.yaml 

    libraries:
#依赖Django 1.5
- name: "django"
  version: "1.5"

  4. 在项目根目录创建应用的入口文件index.wsgi

    import sae
#这里直接使用Django自动生成wsgi脚本
from lql import wsgi
application = sae.create_wsgi_app(wsgi.application)

  5. Push代码

    #在你应用的git代码目录里，添加一个新的git远程仓库 sae
$ git remote add sae https://git.sinacloud.com/lqlcn
#编辑代码并将代码部署到 `sae` 的版本1。
$ git add .
$ git commit -am "init"
$ git push sae master:1
  6. 浏览器查看是否报错lqlcn.applinzi.com
