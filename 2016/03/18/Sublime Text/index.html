Sublime Text使用

Sublime Text的基本使用与常用插件安装。

安装

  • 在线安装
    Ubuntu可以使用apt-get安装

    sudo add-apt-repository ppa:webupd8team/sublime-text-2 
sudo apt-get update
sudo apt-get install sublime-text-dev
sudo apt-get install sublime-text
  • 离线安装
    1. 下载对应平台安装包(Sublime Text 2,Sublime Text 3,)
    2. 双击或使用Ubuntu软件中心安装

基本使用

  • 自动完成：自动完成的快捷键是Tab，如果在html文件中，输入cl按下tab，即可自动补全为class=””；
  • 多列编辑：按住ctrl点击鼠标，会出现多个闪烁的光标，这时可同时修改多处或者按住鼠标中键拖拽；
  • 代码注释功能：ctrl+/、ctrl+shift+/分别为行注释和块注释，再按一下就能去掉注释；
  • 行操作：ctrl+alt+↑、ctrl+alt+↓向上或者向下交换一行；ctrl+enter，光标所在行后插入空行，ctrl+d选择相似；

插件安装

  • Package Control - 包管理组件
    使用Package Control可以很方便的安装其他插件。

    • 在线安装
      使用快捷键`ctrl + ``打开终端;在命令行中输入以下代码来安装Package Control, 等待其自动下载安装；重启Sublime Text

      #Sublime Text 3
import urllib2,os,hashlib; h = '2915d1851351e5ee549c20394736b442' + '8bc59f460fa1548d1514676163dafc88'; pf = 'Package Control.sublime-package'; ipp = sublime.installed_packages_path(); os.makedirs( ipp ) if not os.path.exists(ipp) else None; urllib2.install_opener( urllib2.build_opener( urllib2.ProxyHandler()) ); by = urllib2.urlopen( 'http://packagecontrol.io/' + pf.replace(' ', '%20')).read(); dh = hashlib.sha256(by).hexdigest(); open( os.path.join( ipp, pf), 'wb' ).write(by) if dh == h else None; print('Error validating download (got %s instead of %s), please try manual install' % (dh, h) if dh != h else 'Please restart Sublime Text to finish installation')

#Sublime Text 3
import urllib.request,os,hashlib; h = '2915d1851351e5ee549c20394736b442' + '8bc59f460fa1548d1514676163dafc88'; pf = 'Package Control.sublime-package'; ipp = sublime.installed_packages_path(); urllib.request.install_opener( urllib.request.build_opener( urllib.request.ProxyHandler()) ); by = urllib.request.urlopen( 'http://packagecontrol.io/' + pf.replace(' ', '%20')).read(); dh = hashlib.sha256(by).hexdigest(); print('Error validating download (got %s instead of %s), please try manual install' % (dh, h)) if dh != h else open(os.path.join( ipp, pf), 'wb' ).write(by)

    • 离线安装
      下载Package Control.sublime-package离线安装包，将其放入Sublime Text/Data/Installed Packages目录，重启Sublime Text

    • 用Package Control安装插件的方法

      1. 按下Ctrl+Shift+P调出命令面板
        1. 输入install 调出 Install Package 选项并回车，然后在列表中选中要安装的插件，回车安装。

    • 常用命令
      Disable Package ：禁用插件
      enable Package ：启用插件
      Install Package ：安装插件
      List Package ：查看已安装插件列表
      Remove Package ：移除插件
      Upgrade Package ：升级插件

  • Emmet（Zen Coding）插件
    Emmet插件是一款快速编写HTML代码的工具，使用类CSS语法来缩短代码长度。

    • 基本语法（Emmet很多功能在Sublime中不支持）
      div#id.class1.class2 : 设置div的id和class
      div > child#childId : 设置子元素
      div#div1 + div#div2 : 设置同级元素
      div#id$*3 : 设置三个div，id依次为id1，id2，id3
      a[href=”javascript:void(0);”]{Links} ：设置属性和文本（Links）

    • 快捷键
      ctrl + alt + enter :调出Emmet命令行，输入的语句实时展现
      tab ： 在html中输入emmet语法后，展开语法

    • 注意事项

      • Emmet需要pyV8，插件加载时会自动安装，但因为服务器在国外，经常安装失败，可以离线安装
        1. 下载pyV8对应版本，32bit还是64bit取决于Sublime，而不是操作系统
        2. 解压文件，若是Windows平台则创建文件目录Sublime Text/Data/Packages/PyV8/win32,将解压的文件放入
        3. 重启Sublime

  • JsFormat插件（Format: Javascript）
    JsFormat是一个格式化JS代码的插件，但由于其JSON原生，所以可以用来格式化JSON和其他代码。

    • 快捷键
      ctrl + alt + F ： 格式化选中代码

  • SublimeTmpl插件
    SublimeTmpl插件是新建文件的模板插件，根据文件后缀名自动写入对应模板内容。

    • 自定义模板
      模板文件在/SublimeTmpl/templates中，以后缀名命名，修改对应tmpl文件即可。
    • 快捷键
      ctrl+alt+h ：　新建html文件
      ctrl+alt+j ：　新建javascript文件
      ctrl+alt+c ：　新建css文件
      ctrl+alt+p ：　新建php文件
      ctrl+alt+r ： 新建ruby文件
      ctrl+alt++shift+p 新建python文件
  • SublimeText-Nodejs
    提供nodejsAPI的补全功能和代码提示。
    ps：感觉并不好用
  • less
    支持less的语法高亮。
  • autoprefixer
    提供css前缀自动补充功能，解决兼容性问题。
    • 使用
      ‘ctrl+shift+p’打开命令窗口，输入autoprefixer，即可对当前文件进行自动扩充；或者，手动设置快捷键Preferences > Key Bindings - User添加{ "keys": ["alt+p"], "command": "autoprefixer" }。
  • jQuery
    提供对jquery的api提示。

    其他

  • ctrl + shift + P ：打开控制面板
  • ctrl + ` ：打开终端
  • 离线安装通用方法
    1. 下载插件源代码(https://packagecontrol.io)
    2. 解压源代码放入插件目录(Sublime Text/Data/Packages)
    3. 重启Sublime Text
