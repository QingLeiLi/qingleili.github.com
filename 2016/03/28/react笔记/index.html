React笔记

安装

  • 浏览器

    1. 引入React相关文件

      • react.js：React 的核心库
      • react-dom.js：提供与DOM相关的功能
      • browser.min.js：将JSX语法转为JS语法，很耗时，上线时应提前转为js。

    2. 当引入JSX语法文件时将type设置为text/babel确保可以被解析为js代码。

      <script type="text/babel" src="./test.jsx">

      示例

      <!DOCTYPE html>




      Hello React!








应用场景

  • 复杂场景下的高性能
  • 组件的高重用

JSX

jsx是facebook为适应React而创建的一个语法糖，需要额外的jsx库-browser.min.js来进行解析，最终解析为原生js代码。

jsx的安装

  1. 在HTML页面中引入解析库

    <script src="react/react/build/browser.min.js"></script>

  2. 将包含jsx代码的标签或文件的type设置为’text/babel’

    <script type="text/babel" src="react/react/build/jsx.js"></script>

jsx的使用

  • 使用变量-{}
    {}用于显示变量，而this.props则是渲染时标签的属性对象。

    var Hello = React.createClass({
  render: function() {
    //此处this.props为{'name':'World',}
    return <div>Hello {this.props.name}</div>;
  }
});
ReactDOM.render(
  <Hello name="World" />,
  document.getElementById('container')
);

    jsx注意事项

  • React 的 JSX 里约定分别使用首字母大、小写来区分本地组件的类和 HTML 标签。

  • 由于 JSX 只是是 JavaScript的封装，一些标识符像 class 和 for 不建议作为 XML 属性名。作为替代，React DOM 使用 className 和 htmlFor 来做对应的属性。

    var Hello = React.createClass({
  render: function() {
    //此处定义class时，应使用className
    return <div className='test' >Hello World</div>;
  }
});

  • react中，样式应该使用一个对象表示，而不是字符串;且对象的键应该为属性名的驼峰命名法。

    var styleObject = {
    coclor:'red',
    //此处的font-size属性应写为fontSize
    fontSize:'20px',
}
var Hello = React.createClass({
  render: function() {
    return <div style={styleObject} >Hello World</div>;
  }
});

react的状态与Hook

组件状态

当一个组件（标签）被定义完成后，处在未使用状态；调用renderComponent()进行使用时，组件状态为Mounted状态；调用setState()，setProps()或render()对组件进行更新时，处于Update状态；当调用unmountAndReleaseReactRootNode()时，处于Unmounted状态。

组件状态Hook

  • Mounting状态
    组件开始调用到dom渲染完成，组件都处于该状态。
    该状态可以设置两个Hook-挂载前，挂载后后，定义组件时使用componentWillMount与componentDidMount属性指定。

    var Hello = React.createClass({

    render: function() {
    return <div>Hello {this.props.name}</div>;
},
// 将要mount时的执行函数
componentWillMount:function(){
    alert('willMount');
},
// mount后的执行函数
componentDidMount:function(){
    alert('didMount');
},

    });

  • updating状态
    当组件state被设置时，组件进入该状态，react首先比较新state与旧state是否相同，相同则不作处理，不同则对组件进行更新。
    该状态有四个Hook：
    1. 接受新数据对象-componentWillReceiveProps
      该属性的参数为新的props，可以在该函数内比较新旧props，修改state对象；
    2. 判断是否更新dom-shouldComponentUpdate
      参数为新的props对象和新的state对象，返回true则表示更新dom，false表示不更新dom；
    3. 组件将要更新前-componentWillUpdate
    4. 组件更新完后-componentDifUpdate
  • unmounting状态
    当组件被销毁时进入该状态。
    该状态只有一个Hook函数，componentWillUnmount,可以使用该函数进行一些扫尾操作。

react组件的事件监听

react可以在组件标签上设置ref属性用来找到真实dom节点；
react的事件使用onClick，onChang结构定义，直接写在组件标签上；
react的事件是在js原生事件的基础上封装的，具有原生属性和react拓展的方法，可以使用其阻止冒泡等操作。

var TestClickComponent = React.createClass({
    // 使用一个属性来定义点击事件
    handleClick:function(event){
        // 根据ref找到真实dom节点
        var tip = ReactDOM.findDOMNode(this.refs.tip);
        if(tip.style.display==='none'){
            tip.style.display='inline';
        }
        else{
            tip.style.display='none';
        }
    },
    render: function() {
        return (
            <div>
                // 在此处绑定点击事件
                <button onClick={this.handleClick}>显示|隐藏</button>
                // 此处设置了ref属性，可以使用ReactDOM.findDOMNode(this.refs.tip)来获取真实DOM节点
                <span ref='tip'>测试点击</span>
            </div>
            );
    },
});

props与state

props是指组件被调用时，调用者传递给组件的值，该值一般为不可变的，创建好组件对象后，该属性就伴随该对象了，如果要改变则需要新建对象。

ReactDOM.render(
    //这里的name='World'为props，该变量会被传递给组件，组件使用{this.props.name}来获取
    <Hello name="World" />,
    document.getElementById('container')
);

state为组件定义时，内部使用的变量，可以被更改来改变组件各种属性或状态，组件使用{this.state.name}来获取。

var Hello = React.createClass({
    // 使用getInitialState属性定义state，返回一个对象
    getInitialState:function(){
        alert('init');
        return {
            opacity:1.0,
            fontSize:'12px',
        }
    },
    render: function() {
        return <div style={this.state}>Hello World</div>;
    },
    // 在组件挂载后可以更改state
    componentDidMount:function(){
            alert('did');
            // 在此将透明度改为0.5，组件会进入update状态，更新dom
            this.setState({
                opacity:0.5,
            });
        },
});
