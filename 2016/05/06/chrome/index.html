Chrome笔记

记录一些平时使用到的技巧或命令。

调试

###　事件

  • 使用monitorEvents()与unmonitorEvents()查看触发的事件
    monitorEvents()与unmonitorEvents()是chrome的调试命令，在开发者工具的终端输入即可，前者用来监听某个dom上面的所有指定类型事件；后者用来解除绑定。
    eg： 

    monitorEvents(document.body, 'mouse')  
监听body上，所有的鼠标事件，发生的事件会在终端打印出来  
unmonitorEvents(document.body,'mouse')  
解除监听body的鼠标事件

    事件列表： 

    mouse:  “mousedown”, “mouseup”, “click”, “dblclick”, “mousemove”, “mouseover”, “mouseout”, “mousewheel” 
key: “keydown”, “keyup”, “keypress”, “textInput”
touch:  “touchstart”, “touchmove”, “touchend”, “touchcancel”
control:  “resize”, “scroll”, “zoom”, “focus”, “blur”, “select”, “change”, “submit”, “reset”
no arguments: all of the above + “load”, “unload”, “abort”, “error”, “select”, “change”, “submit”, “reset”, “focus”, “blur”, “resize”, “scroll”, “search”, “devicemotion”, “deviceorientation”

  • 使用VisualEvent拓展查找事件绑定函数
    chrome自带的事件查看器显示jquery等框架绑定的事件函数时，总是定位到jquery，无法定位到事件函数，使用VisualEvent可以查看dom上绑定的所有事件函数。
    VisualEvent安装后在拓展栏是一个小眼睛，点击后，他会将所有绑定了事件的dom节点用不同的颜色标记，鼠标移动到dom上时，会显示绑定的事件以及事件函数的定义位置。

  • js事件断点
    在开发人员工具-source右侧的Event Listener Breakpoints可以对指定类型的事件下断点，并使用F8和F10来继续运行或单行调试。

代理

  • SwitchySharp代理
    SwitchySharp 是 Google Chrome 浏览器上的一个代理管理扩展程序，可以用来搭梯子。
    安装后对其进行配置：
    1. 下载SwitchyOptions.bak配置文件
    2. 打开SwitchySharp配置界面，选择导入导出中从配置文件导入，该备份文件会设置好GoAgent，Shadowsockes，GoAgent PAC和shadowsocks PAC四个情景模式；
    3. 选择任意一个模式进行配置，修改代理端口

报文

  • Header Hacker拓展
    Header Hacker是一个修改http请求报文头的插件，可以很容易的定制请求头。
  • Edit This Cookie拓展
    Edit This Cookie是一个cookies编辑拓展。
  • ColorZilla
    页面拾色器。

    应用

  • Postman
    Postman是一个模拟HTTP请求的应用，支持HTTP的所有请求方法和auth认证等，可以很方便的用来调试RESTFul API。

其他

  • Video-Downloader-professional
    一个媒体下载插件，可以使用其下载页面上的视频。
  • Octotree
    用于github和gitlab进行代码浏览的侧边栏插件，可以很方便的查看目录结构并定位文件。
