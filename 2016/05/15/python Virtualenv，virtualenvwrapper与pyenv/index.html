python Virtualenv，virtualenvwrapper与pyenv

virtualenv是python的虚拟沙盒，而virtualenvwrapper是沙盒的管理工具；pyenv是python的版本管理，可以切换全局/局部python版本。

virtualenv

virtualenv是python的虚拟沙盒，可以建立多个沙盒环境用于不同的项目进行开发，沙盒中的软件包是不会相互影响的，甚至python版本也可以不同，但是使用沙盒的时候是shell级别的，即某一个沙盒激活后，仅仅是当前shell运行在该沙盒中，shell关闭后失效；仅仅是方便多套环境下的开发。

安装

  • pip

    pip install virtualenv

使用

  1. 创建虚拟环境 

    # 建立一个名为test_env的虚拟环境
$ virtualenv test_env
New python executable in /home/lql/Desktop/test_env/bin/python
Installing setuptools, pip, wheel...done.

    在创建的沙盒中会有python的一个副本，并安装好了setuptools, pip。
    注意：老版本的virtualenv默认会将全局包加入虚拟环境，需要的话使用--no-site-packages参数指定不引入系统已安装的第三方包；新版本默认不会引入系统包，需要的话使用--system-site-packages参数指定其引入系统包。

    virtualenv --no-site-packages ENV
virtualenv --system-site-packages ENV

  2. 激活该沙盒 

    $ cd test_env/
# 激活沙盒
$ source ./bin/activate
# 激活后在命令行前会提示当前激活的沙盒名
(test_env)$
  3. 安装所需软件包
    沙盒激活后安装的软件包都安装在沙盒中，不与其他沙盒或全局软件包冲突，可以定制项目开发环境。

  4. 退出虚拟环境

    使用deactivate命令退出当前沙盒，该命令可以在任意目录执行。

    $ deactivate

    不需要的沙盒可以直接rm删除。

virtualenvwrapper

virtualenvwrapper是一个建立在 virtualenv 上的工具，通过它对沙盒进行管理，可以方便的创建/激活/管理/销毁虚拟环境。

###安装

  1. 使用pip安装拓展包

    pip install virtualenvwrapper

  2. 配置脚本
    软件包安装完后会创建virtualenvwrapper.sh脚本，位置貌似不同系统不一样，查找到脚本位置。

    $ which virtualenvwrapper.sh
/usr/local/python2.7/bin/virtualenvwrapper.sh

    virtualenvwrapper需要配置环境变量才能正常工作，将配置脚本写入~/bashrc.，开启终端自动运行，在文件最后追加

    # 判断脚本是否存在
if [ -f /usr/local/python2.7/bin/virtualenvwrapper.sh ]; then
    # virtualenvwrapper存放沙盒文件的位置
    export WORKON_HOME=$HOME/.virtualenvs
    # 运行virtualenvwrapper脚本初始化
    source /usr/local/python2.7/bin/virtualenvwrapper.sh
fi

    使其马上生效

    source ~/.bashrc

    这样就可以使用virtualenvwrapper了。

使用

  • mkvirtualenv：创建新的沙盒，该沙盒会被创建在环境变量WORKON_HOME指定的位置，生成沙盒与virtualenv一致

    $ mkvirtualenv test
New python executable in /home/lql/.virtualenvs/test/bin/python
Installing setuptools, pip, wheel...done.
virtualenvwrapper.user_scripts creating /home/lql/.virtualenvs/test/bin/predeactivate
virtualenvwrapper.user_scripts creating /home/lql/.virtualenvs/test/bin/postdeactivate
virtualenvwrapper.user_scripts creating /home/lql/.virtualenvs/test/bin/preactivate
virtualenvwrapper.user_scripts creating /home/lql/.virtualenvs/test/bin/postactivate
virtualenvwrapper.user_scripts creating /home/lql/.virtualenvs/test/bin/get_env_details

  • workon：激活沙盒

    $ workon test
(test)$

  • deactivate：退出沙盒

    (test)$ deactivate 
$

  • rmvirtualenv：删除沙盒

    $ rmvirtualenv test
Removing test...
$

  • lssitepackages：查看沙盒中的包列表，必须有沙盒处于激活状态

    (test)$ lssitepackages
easy_install.py   pip-8.1.2.dist-info  setuptools-21.0.0.dist-info
easy_install.pyc  pkg_resources        wheel
pip               setuptools           wheel-0.29.0.dist-info

pyenv

pyenv是python的版本控制，可以在全局切换python版本，而不是使用沙盒。

安装

  • 自动脚本
    依赖git，默认安装在~/.pyenv

    curl -L https://raw.githubusercontent.com/yyuu/pyenv-installer/master/bin/pyenv-installer | bash

    将pyenv加入环境变量，~/.bashrc追加一下内容

    export PATH="/home/lql/.pyenv/bin:$PATH"
eval "$(pyenv init -)"
eval "$(pyenv virtualenv-init -)"

使用

  • pyenv rehash:将本机安装的python版本纳入pyenv控制

  • pyenv install 2.7.6：安装新的python版本

    $ pyenv install 2.7.6
$ pyenv versions
  system
  2.7.6
  • pyenv install –list：查看已安装的python列表
  • pyenv uninstall：卸载一个python版本
  • pyenv local 2.7.6：设置当前文件夹python版本为2.7.6
  • pyenv global 2.7.6：设置系统全局python版本为2.7.6
  • pyenv shell 2.7.6：将当前shell的python版本设置为2.7.6
